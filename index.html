Explore Pixodesk Animator features

Pixodesk Animator simplifies the creation of animated content by combining a powerful vector editor and animator while also offering a wide range of exporting options.
The app is built upon the popular Lottie file format, offering you increased flexibility for using it with various other drawing and rendering tools.
There is also a vast and active Lottie artwork community that you can tap into when crafting your own artwork.
Create
  • Create vector drawings, import vector or raster images, use custom ttf fonts.
  • Reuse and modify pre-existing animations from Lottie community, e.g. from here LottieFiles .
  • Compose complex animated scenes by using animation blocks.
Animate
  • Create keyframe based animation in a traditional way by using "watch" and "diamond" buttons. Create animation loops, blocks and time-remaps.
  • Automate the animation of objects on the page by freezing object states at one or more time points and making alterations at a different, non-frozen time point.
  • Select a pre-defined easing function from a dropdown menu or customize the easing curve by editing it. Easing functions come with velocity and acceleration charts that enhance your comprehension of animation dynamics.
Export
  • Export a compact, animated SVG file for the web, using either JS or CSS animation types. You can then embed this SVG directly into your webpage, ensuring it looks good on screens with different resolution.
  • Control how the animation starts and ends on your webpage: on click, scroll, mouse in/out.
  • Use your Lottie file with the Lottie rendering libraries to use in your native applications on Windows, iOS, Android, and ReactNative platforms.
  • Export to other media formats: webm, gif, png, svg.
Auto-animate
Automate the animation of objects on the page by freezing object states at one or more time points and making alterations at a different, non-frozen time point.
Easing
  • Select a pre-defined easing function from a dropdown menu or customize the easing curve by editing it. Easing functions come with velocity and acceleration charts that enhance your comprehension of animation dynamics.
  • Keyframe segments are visually represented as mini velocity charts, providing you with insights into the easing dynamics.
Create animated icons
  • Draw or import an existing icon, and effortlessly animate it using options like Trim Path, Dash Array, Rotation, Scale, Skew, and more. Achieve various effects such as fading, flipping, bouncing, stroke disappearing, and stroke moving along a path.
  • Use grid and snap modes to achieve precise and accurate drawing.
  • Use your animated icon on your website or mobile/desktop application.
Create animated splash-screens
  • Create visually stunning splash-screen animations to engage with users, giving them a good first impression.
