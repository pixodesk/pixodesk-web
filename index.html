Lottie animation file
  • JSON-based animation file format
  • Allows to have vector, image and text content
  • Resolution independent
  • Frame-rate independent
  • Keyframe based
Lottie World
  • Big lottie community.
  • Lottie documentation.
  • Collection of existing animations at LottieFiles.
For developers
  • Platform-independent.
  • Open-source and free libraries that play Lottie files exist for many platforms: Web, iOS, Android, Windows, React Native. r-lottie.
Use for
  • Animated icons, loaders, splash screens, stickers.
  • Figma, Webflow, Elementor have support for Lottie files.

Pixodesk Animator features

  • Works with Lottie animation file natively
  • Brings all tools together speeding up your work
  • Easier to create more complex animation
  • Easier to reuse animation

Create & Reuse
  • Create vector drawings.
  • Reuse existing Lottie files. Add, change or compose animations.
  • Reuse animation blocks multiple times.
  • Import existing Lottie animations into the project.
  • Compose by adding existing vector or raster images (svg, png, jpeg, webp).
  • Import and use custom fonts.
  • Use browser fonts or glyphs.
Animate: Auto animation
  • Automatically animate objects on the page by freezing object states at one or multiple time points and changing something at another unfrozen time point.
  • Keyframe based animation.
  • Make element property animated by clicking on a clock icon next to the option.
Animate: Easing Charts
  • Choose predefined easing function (accompanied by velocity and acceleration charts) from a dropdown or edit the easing curve.
  • Keyframe segments/ranges are shown as velocity mini charts that allow you to see what kind of easing is going on there.
  • The animation timeline can be represented as value, velocity and acceleration charts.
Export
  • Export static vector (*.svg).
  • Export raster images (*.png, *.jpeg, *.webp).
  • Save as Lottie animation file (*.json, *.lottie).
  • Export SVG+CSS animation.
  • Export animated GIF.
  • Export video (*.webm).
  • Allows export big high quality animation in small size.
  • Improves page load speed.
Create splash screens
  • Create visually stunning splash-screen animations to engage with users giving them a good first impression.
  • Use animation blocks, auto-keyframes, text, mask, clipping, gradient, along-path and other tools.
Create animated icons
  • Draw or import existing icon and easily animate it with such options as Trim path, Dash Array, Rotation, Scale, Skew, etc. to achieve such effects as fading, flipping, bounce, stroke disappearing, stroke moving along path.
  • Use grid and snap modes to achieve precise and accurate drawing.
  • Use your animated icon on your web site or mobile/desktop application.
