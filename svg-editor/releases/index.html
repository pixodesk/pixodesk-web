Pixodesk SVG Editor - Release Notes

Version 1.8 - August 2024

  • Added Round Corner tool
  • Added Spray tool
  • Bug fixes

Version 1.7 - June 2024

  • Added a new type of color picker
  • Added more snapping modes
  • Bug fixes

Version 1.6 - April 2024

  • Added Shape Builder tool
  • Added Star and Polygon tool
  • Added Shift hotkey to lock star and polygon rotation to 45 degrees when adding them
  • Added Clone / Repeater tool for SVG element
  • Added option to align the Anchor Point in the shape: to the center, to the corners, or to the middle of its sides
  • Added the element lock feature to prevent elements from being interactable on the canvas
  • Bug fixes

Version 1.5 - March 2024

  • Added text along path
  • Improved attribute UI
  • Added SVG filters
  • Improved bezier control points
  • Bug fixes

Version 1.4 - February 2024

  • Initial release
